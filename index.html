Instructions to Dress Up Your Blog

Feb 10th, 2015 6:02 pm

I learn these instructions from “Hailong Hao’s Laboratory”

When you set down your home blog, you start to think about adding some furnitures, painting walls, decoration , and so on.

  • Use some third party themes to replace the default Classic. Such as the theme cleanpress
cd octopress
git clone git://github.com/macjasp/cleanpress.git .themes/cleanpress
rake install['cleanpress']
rake generate
  • Configure your basic settings in _config.yml file,
    • Domain name, Blog Title, Author
url: http://yoursite.com  #这里改为你网站的域名
title: My Octopress Blog #这里改为你想要的网站的标题
subtitle: A blogging framework for hackers, #这里改为你的博客副标题
author: Your Name #这里改为博客作者的名字，也就是你的名字
simple_search: https://www.google.com/search #这里是默认搜索引擎，可以先顾不管
description: #网站描述信息
  • Time & Date format

Because this is a English default blog, if you want to use Chinese Time/Date format in your blog. you can edit this part

date_format: "ordinal" #默认日期显示方式

Please modify ordinal to %Y 年%-m 月%-d in _config.yml

* link

Change the default link

http://[your_domain_name]/blog/2014/04/28/Post-Title/ ＃这是自动生成的博客链接

to

http://[your_domain_name]/blog/20140428/Post-Title.html/
http://[your_domain_name]/blog/Post-Title.html/ # 这条链接清晰明了

in the _config.yml file, you can find

permalink: /blog/:year/:month/:day/:title/ # 文章固定链接

you can change it into the following format

permalink: /blog/:year:month:day/:title.html/

or

permalink: /blog/:title.html/ # this is the simplest style
  • Change “Categories” to “分类” ＃修改分类前缀

Just add the following line into _config.yml

category_title_prefix: "分类：" ＃ 修改分类前缀

you can add this line in any position of _config.yml file. or after

category_dir: blog/categories
  • [Continue] Button

excerpt_link: “Read on &raar;” # “Continue reading” link text at the bottom of excerpted articles

Change “Read on” to “继续阅读”

rake generate
rake deploy

Instructions to Post on Ocotpress

Instructions to post on Octopress

When you have installed and built your octopress blog environment, you can start to post somehting on your blog.

Enter octopress

cd octopress

Create a new entry

rake new_post["Post Title"]

Please replace Post Title with your own title, this will create a markdown file named yyyy-mm-dd-Post-Title.markdown, next you can edit your blog content in this file.

Then go to _posts directory to edit your new file

cd source/_posts/
vim yyyy-mm-dd-Post-Title.markdown

When you finish editing your markdown file, use the following comment to post on your octopress blog

rake gen_deploy
