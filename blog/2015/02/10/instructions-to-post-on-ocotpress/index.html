Instructions to post on Octopress

When you have installed and built your octopress blog environment, you can start to post somehting on your blog.

Enter octopress

cd octopress

Create a new entry

rake new_post["Post Title"]

Please replace Post Title with your own title, this will create a markdown file named yyyy-mm-dd-Post-Title.markdown, next you can edit your blog content in this file.

Then go to _posts directory to edit your new file

cd source/_posts/
vim yyyy-mm-dd-Post-Title.markdown

When you finish editing your markdown file, use the following comment to post on your octopress blog

rake gen_deploy
